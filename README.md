# 顧客一覧アプリ

## ファイル構成
```
顧客一覧アプリ/
├── proxy-server.js   ← プロキシサーバー
├── 顧客一覧.html     ← アプリ本体
└── README.md         ← この説明書
```

## 使い方

### 1. プロキシサーバーを起動
ターミナルでこのフォルダに移動して実行：
```bash
cd ~/Desktop/顧客一覧アプリ
node proxy-server.js
```
`✅ プロキシサーバー起動中` と表示されればOK

### 2. アプリを開く
`顧客一覧.html` をダブルクリック → ブラウザで開く

### 3. 接続情報を入力
| 項目 | 値 |
|---|---|
| シークレットID | 発行されたID |
| シークレットパスワード | 発行されたパスワード |
| データベースID | nagaoka_customer |
| データベースパスワード | customer |
| テーブル名 | nagaoka_official |

### 注意
- プロキシサーバー（ターミナル）は起動したまま閉じないでください
- PCを再起動したら再度 `node proxy-server.js` を実行してください
